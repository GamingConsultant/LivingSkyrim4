## SSE Display Tweaks
1. Enable FPS limit on main screen  
	```
	UIFramerateLimitMain=60
	UIFramerateLimitMainVSyncOff=false  
	```
2. Maybe help with stutters a bit
	```
	[HAVOK]
	MinimumFramerate=40
	MaximumFramerate=120
	```
	
## INI tweaks
1. Smooth out zooming  
	```
	[Camera]
	fMouseWheelZoomSpeed=1.00 
	```
